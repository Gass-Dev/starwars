# starwars

## FrontEnd Application Desktop
Next.js

https://nextjs.org/

## Api
https://swapi.dev/

## Commands for Next.Js
### Starts the development server:
cd starwars
yarn dev

### Builds the app for production:
 yarn build
 
### Runs the built app in production mode:
yarn start
